# ミディ
バッハのゴールドベルク変奏曲を演奏するミディシンセサイザー。

前のトラック: 左矢印キー
次のトラック: 右矢印キー
速度調整: 上下矢印キー
一時停止: スペースキー

高音を一番上にした88鍵のピアノ音符のスクロール表示。再生は自動的にリストの次の曲に進みます。

[ゴールドベルク変奏曲](https://ja.wikipedia.org/wiki/ゴルトベルク変奏曲)
