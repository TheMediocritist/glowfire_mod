# maze3d
Find the end. You'll know when you find it ;)
Move using arrow keys.

On SQFMI beepy:
* Press menu button above keyboard.
* Keyboard will enter arrow key mode.
* See the star icon at the top right of the screen.
* E-S: forward, back.
* W-D: left, right.
